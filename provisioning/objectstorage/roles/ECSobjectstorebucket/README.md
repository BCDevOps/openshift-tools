Role Name
=========

ECSobjectstorebucket

Role to provision/deprovision an ECS object bucket, and an object user (owner) of the bucket.  This includes provisioning appropriate k8s objects as per Object Bucket Claim (OBC) design.

Requirements
------------

Role uses the EC2 module, the boto package is required.
